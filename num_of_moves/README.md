For inputArray = [1, 1, 1], the output should be
arrayChange(inputArray) = 3.

Input/Output
3 ≤ inputArray.length ≤ 105,
-105 ≤ inputArray[i] ≤ 105.

var a = [1,1,1];
console.log(arrayChange(a));
